# 📊 Product Line Profitability Analysis

## 🔍 Overview
This project analyzes product-level profitability and margin performance for a distribution business.

## 🚀 Features
- KPI Dashboard
- Product Profitability Analysis
- Division Performance
- Cost vs Sales Diagnostics
- Pareto Analysis (80/20 Rule)
- Risk Classification

## 🛠 Tech Stack
- Python
- Pandas
- Streamlit
- Plotly

## ▶️ Run Locally
```bash
streamlit run app.py
